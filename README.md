Pasos rápidos para publicar en GitHub (sin conocimientos avanzados):

1. Crea una cuenta en https://github.com si no tienes una.
2. Crea un nuevo repositorio (por ejemplo: atm-seguros). Marca "Public".
3. Subir los archivos: puedes arrastrar y soltar los archivos desde la carpeta o usar Git (comandos abajo).
4. Habilitar GitHub Pages: Repository -> Settings -> Pages -> Source: Branch 'main' (root) -> Save.
5. Espera unos minutos y tu sitio estará disponible en: https://TU_USUARIO.github.io/NOMBRE_REPO

Comandos Git (si prefieres terminal):

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/TU_USUARIO/NOMBRE_REPO.git
git push -u origin main
